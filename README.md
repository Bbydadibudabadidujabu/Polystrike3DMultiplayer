# Polystrike3DMultiplayer
New Game -> https://bbydadibudabadidujabu.github.io/Polystrike3DMultiplayer/
WASD - Move
Space - Jump
Left Click - Shoot
To host an game u need to type ur username and press multiplayer and host the game and for players to join you need to look at the top right corner of your game and it shows the code so people can join. to join you enter your username and write down the 5 letter long code in the multiplayer and you press join server you joined to the hosts server.
to activate ur abilities u press esc to unlock your mouse and press the ability u want. you can only choose one every cooldown. max 5 players in 1 server.
